# 📸 GourmetLens AI (美食镜头 AI)

> **English** | [中文版](README_zh-CN.md)

> **面向全球的虚拟美食摄影师。**  
> 利用 Google 最新的生成式 AI 模型，即时为菜单和社交媒体生成专业级的高端美食摄影作品。

![应用截图](ScreenShot_2025-11-21_064712_937.png)

## 🌍 全球版 (支持 54 种语言)

GourmetLens AI 专为全球用户设计。它将其界面和烹饪知识库完全适配到了 **54 种不同的语言**，让从东京到里约热内卢的用户都能用母语描述美食。

**核心功能：**
*   **54 种语言支持：** 完整的 UI 翻译，包括对阿拉伯语、希伯来语、波斯语和乌尔都语的 RTL（从右到左）布局支持。
*   **全球美食预设：** 包含全球各地的标志性美食（如 *寿司、塔可、海鲜饭、汉堡*）的精选列表，并针对每种支持的语言进行了文化语境化和翻译。
*   **风格控制：** 在 **复古/暗调 (Rustic/Dark)**、**现代/明亮 (Bright/Modern)** 和 **社交媒体/俯拍 (Social Media)** 风格之间自由切换。

## 🤖 Google AI 驱动

本应用利用了 **Google GenAI SDK** 的尖端能力：

*   **图像生成：** 使用 **`imagen-4.0-generate-001`** 模型创建超逼真的食物质感、光影和摆盘效果。
*   **魔法编辑器：** 使用 **`gemini-2.5-flash-image`** 模型理解自然语言的编辑请求（例如：*“增加升腾的热气”、“让灯光更暖一点”*），并即时修改现有图像。

## 🚀 快速开始

### 前提条件

*   已安装 Node.js。
*   拥有一个 Google AI Studio API Key，且具备 `imagen-4.0` 和 `gemini-2.5-flash` 的访问权限。

### 安装步骤

1.  **克隆仓库**
    ```bash
    git clone https://github.com/skaiy/GourmetLens-AI.git
    cd gourmetlens-ai
    ```

2.  **安装依赖**
    ```bash
    npm install
    ```

3.  **配置环境**
    在根目录下创建一个 `.env` 文件：
    ```env
    API_KEY=your_google_ai_studio_api_key_here
    ```

4.  **运行应用**
    ```bash
    npm start
    ```

## 📖 使用指南

1.  **选择语言：** 点击右上角的地球图标，选择 54 种支持语言中的一种。
2.  **选择预设（可选）：** 点击“快速开始”下的美食按钮（例如：🍕 Pizza, 🍣 Sushi），应用会自动以您选择的语言填入专业撰写的提示词。
3.  **自定义：**
    *   **菜名：** 例如：“黑松露蘑菇炖饭”。
    *   **描述：** 描述食材、氛围和光线。
    *   **风格：** 选择复古、现代或社交媒体风格。
4.  **生成：** 点击“生成照片”。
5.  **编辑：** 点击任何生成的图片打开 **魔法编辑器**。输入指令如 *“去掉餐巾”* 或 *“加一杯红酒”* 来微调照片。

## 🌐 支持语言

GourmetLens AI 支持以下地区语言：

*   **英语** (美国, 英国)
*   **东亚：** 中文 (简体/繁体), 日语, 韩语
*   **欧洲：** 西班牙语, 法语, 德语, 意大利语, 葡萄牙语 (葡萄牙/巴西), 俄语, 荷兰语, 波兰语, 乌克兰语 等。
*   **中东：** 阿拉伯语, 希伯来语, 波斯语, 土耳其语。
*   **南亚：** 印地语, 孟加拉语, 马拉地语, 泰米尔语, 泰卢固语, 卡纳达语, 古吉拉特语, 马拉雅拉姆语。
*   **东南亚：** 越南语, 泰语, 印尼语, 马来语, 菲律宾语。
*   **非洲：** 斯瓦希里语, 阿姆哈拉语。
